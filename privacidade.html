Política de Privacidade

Entenda quais dados o Data Skill Map coleta, por que eles são usados e como você pode solicitar alterações ou exclusão.

Última atualização: 20 de junho de 2026.

O Data Skill Map é uma plataforma educacional criada para ajudar pessoas a identificarem seu nível em dados, receberem uma trilha de estudo personalizada e praticarem habilidades como lógica de dados, SQL, estatística, Excel, indicadores e projetos práticos.

Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos informações relacionadas ao uso da plataforma.

1. Quem somos

O Data Skill Map é uma plataforma de aprendizado em dados disponível em: https://trilhadedados.com.br

Contato para dúvidas sobre privacidade, acesso, correção ou exclusão de dados: admin.trilhadedados@gmail.com

2. Quais dados podemos coletar

Podemos coletar os seguintes dados, conforme o uso da plataforma:

  • nome e e-mail, quando você cria uma conta ou entra com Google;
  • identificador interno de usuário;
  • respostas do diagnóstico;
  • quantidade de acertos, erros e percentual de desempenho;
  • área de maior dificuldade e trilha recomendada;
  • progresso nas trilhas e práticas;
  • tentativas em exercícios práticos;
  • feedbacks enviados pelo usuário;
  • dados técnicos básicos de navegação, como data de acesso, página acessada e eventos de uso da plataforma.

Não solicitamos intencionalmente dados sensíveis, como informações de saúde, religião, opinião política, origem racial ou étnica, biometria ou dados financeiros.

3. Como usamos os dados

Usamos os dados para:

  • permitir login e identificação da conta;
  • salvar o resultado do diagnóstico;
  • recomendar uma trilha de estudo;
  • acompanhar progresso do usuário;
  • liberar ou organizar práticas educacionais;
  • melhorar perguntas, trilhas, conteúdos e experiência da plataforma;
  • analisar erros, abandono e pontos de dificuldade;
  • receber feedbacks e melhorar a qualidade do produto;
  • manter a segurança e funcionamento da aplicação.

4. Login com Google

Quando você escolhe entrar com Google, usamos o Google OAuth para autenticação. Nesse processo, podemos receber informações básicas da sua conta Google, como nome e e-mail, apenas para criar ou acessar sua conta no Data Skill Map.

O Data Skill Map não acessa sua senha do Google.

5. Onde os dados são armazenados

Os dados da plataforma são armazenados em serviços de infraestrutura e banco de dados utilizados pelo projeto, incluindo Supabase para autenticação, banco de dados e gerenciamento de usuários.

6. Compartilhamento de dados

Não vendemos dados pessoais.

Podemos compartilhar dados apenas com provedores técnicos necessários para funcionamento da plataforma, como serviços de autenticação, hospedagem, banco de dados e infraestrutura.

Também podemos usar dados agregados ou anonimizados para análise de uso e melhoria da plataforma, sem identificação direta do usuário.

7. Cookies e tecnologias semelhantes

A plataforma pode utilizar recursos técnicos necessários para login, sessão, segurança, funcionamento da aplicação e análise básica de uso.

8. Por quanto tempo mantemos os dados

Mantemos os dados enquanto a conta estiver ativa ou enquanto forem necessários para funcionamento, segurança, melhoria da plataforma ou cumprimento de obrigações aplicáveis.

O usuário pode solicitar a exclusão dos seus dados pelo e-mail de contato informado nesta política.

9. Direitos do usuário

Você pode solicitar:

  • confirmação sobre tratamento de seus dados;
  • acesso aos dados associados à sua conta;
  • correção de dados incompletos ou desatualizados;
  • exclusão de dados pessoais, quando aplicável;
  • informações sobre o uso dos dados;
  • revogação de consentimento, quando o tratamento depender de consentimento.

Para exercer esses direitos, envie uma solicitação para: admin.trilhadedados@gmail.com

10. Segurança

Adotamos medidas técnicas e organizacionais razoáveis para proteger os dados contra acesso não autorizado, alteração, perda ou uso indevido.

Mesmo assim, nenhum sistema digital é totalmente imune a riscos. Caso identifiquemos incidente relevante, tomaremos as medidas cabíveis.

11. Uso por menores de idade

A plataforma tem finalidade educacional. Caso um responsável legal identifique uso indevido ou deseje solicitar exclusão de dados de menor de idade, poderá entrar em contato pelo e-mail: admin.trilhadedados@gmail.com

12. Alterações nesta política

Esta política pode ser atualizada para refletir melhorias na plataforma, mudanças legais ou ajustes operacionais. A data da última atualização será sempre exibida no início da página.

13. Contato

Para dúvidas, solicitações ou assuntos relacionados à privacidade: admin.trilhadedados@gmail.com
